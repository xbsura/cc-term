Instant Terminal Sharing

One command to expose any tmux session over the web. Share a URL, open on your phone. Done.

▦

Multi-Session Dashboard

See all your sessions in one page. Click to connect. Switch panes. Split views. All in the browser.

⚡

Zero Config

No accounts. No signup. No API keys for the terminal itself. Install, run, done. Close it when you're done.

📱

Mobile-First Web UI

Responsive terminal with virtual keys, gesture support, and optimized touch layout. Works on any device.

🔒

Session Protection

Optional username/password per session. Private sessions hidden from the global page. You control access.

🌐

Tunnel Through NAT

Built-in reverse tunnel to ttyd.ink. Works behind firewalls, no port forwarding needed.

⚙

Built for Claude Code

Pre-configured tmux, vim, and bash. Session persistence, auto-save, and restore. Optimized for AI workflows.

📦

Fully Self-Contained

Isolated under ~/.cc-term. Own Python venv, own tmux config, own plugins. Won't touch your existing setup.

// how it works
Three commands. That's it.

Install

Run the installer. It sets up brew dependencies, tmux, vim, and the cc-term toolkit.

curl -fsSL https://ttyd.ink/install | bash

Launch a session

Start a new terminal session or attach to an existing one. Named sessions persist across restarts.

cc-term  or  cc-term myproject

Go remote

Inside any session window, enable remote access. Get a URL, scan the QR code, open on your phone.

cc-term -r
